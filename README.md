# Dashboard de Performance de Vendas e Lucratividade (Bootcamp DIO Data Analytics)

[cite_start]Este projeto foi desenvolvido com o objetivo de transformar dados brutos de transações comerciais em um relatório interativo no **Power BI**[cite: 4, 105]. [cite_start]O dashboard permite a exploração de métricas financeiras cruciais, como volume de vendas, margem de lucro e unidades vendidas, permitindo uma análise granular por produto, geografia e segmento de cliente[cite: 105, 107].

## 📊 Visão Geral do Projeto
O relatório está estruturado em três perspectivas principais para facilitar a tomada de decisão estratégica:

1.  [cite_start]**Análise de Produtos e Preços**: Identificação dos itens com maior volume de vendas, destacando o produto **Paseo**, que representa 29,35% do total[cite: 8, 10]. [cite_start]A análise também correlaciona a média de preço de venda por categoria[cite: 14].
2.  [cite_start]**Geolocalização e Lucro**: Monitoramento da performance por país, revelando que a **França** é o mercado mais rentável (22,38% do lucro total), seguida de perto pela Alemanha e Canadá[cite: 82, 90, 93, 94].
3.  [cite_start]**Segmentação de Mercado**: Visualização da distribuição de lucro por perfil de cliente, onde o segmento governamental (**Government**) demonstra dominância absoluta, representando 65,04% do resultado global[cite: 107, 110, 111].

## 🛠️ Ferramentas e Tecnologias
* [cite_start]**Power BI**: Criação de visualizações interativas e dashboards dinâmicos[cite: 4, 105].
* [cite_start]**Linguagem DAX**: Implementação de medidas calculadas para Soma de Vendas, Lucro Total e Médias de Preço[cite: 5, 14, 82].
* [cite_start]**Modelagem de Dados**: Estruturação de dados temporais abrangendo os anos de 2013 e 2014[cite: 1, 60].
* [cite_start]**Mapas de Dados**: Utilização de recursos visuais geográficos para análise de performance regional[cite: 115, 120].

## 📈 Principais Insights Extraídos
* [cite_start]**Sazonalidade**: O relatório identifica um pico de lucratividade em dezembro de 2014, atingindo a marca de 2 milhões, o que sugere uma forte tendência de vendas de final de ano[cite: 60, 61, 80].
* [cite_start]**Eficiência por País**: Embora os Estados Unidos e o Canadá possuam volumes de vendas expressivos, a França apresenta a melhor conversão em lucro real para a organização[cite: 82, 86, 95].
* [cite_start]**Foco Estratégico**: A dependência do segmento governamental (65,04% do lucro) indica uma oportunidade de expansão nos segmentos de *Small Business* e *Midmarket* para diversificação de receita[cite: 107, 110, 112].
